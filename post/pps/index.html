Pursue Path & Separate

Keeping the bots corralled with sheep-bots.

This project implements pursue, path-following, and separation algorithms from Ian Millington’s. The rings of bots have a leader using waypoint-based path following with the bots behind them running pursue code. The outer ring has two leaders with one for the inner. The bots inside these rings run only separation logic. Sheep-bots are avoiding all units such that if they approach the ring while a patrol is there, they should bounce back into the field.
