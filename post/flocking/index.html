Flocking

Flocking using a blended steering algorithm of separation, cohere, and movement-look logic.

An early project in the class, implementing a flocking algorithm from Ian Millington’s AI for Games. This implemented both priority steering for obstacle avoidance priority, and blended steering for separation, coherence, and look-where-going movement algorithms. The boids in this project were also a test of vertex animated textures.
