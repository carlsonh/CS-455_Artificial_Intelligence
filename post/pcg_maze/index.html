PCG - Maze

At dev time, generate a maze using depth first backtracking and have Houdini Engine generate corresponding geometry.

Using a depth first backtracking algorithm implemented from Ian Millington’s AI for Games, this generates a maze of arbitrary size set by the developer. This is generated as a 2-D array of vector4’s representing whether a given direction is open or if there’s a wall there. This is converted to an array of points with the paths stored in the uv’s to be sent to Houdini Engine.

In Houdini Engine, a wall asset is then copied onto each of the directions that should be blocked, the maze model polyreduced a bit, and returned to Unity.
